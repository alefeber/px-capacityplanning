Capacity Counter — T-Shirt Calculator

How to use click to collapse
  1. Select a Person.
  2. Click XS/S/M/L/XL to add work (hours auto-deduct).
  3. Set PTO days (per person) to reduce their capacity.
  4. PTO uses each person’s own daily rate (their sprint capacity ÷ 10).
  5. Use Undo/Clear person/Reset all as needed.
Defaults: XS 2h • S 6h • M 12h • L 22h • XL 32h
